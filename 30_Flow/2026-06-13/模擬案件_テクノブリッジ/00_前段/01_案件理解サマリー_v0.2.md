# 案件理解サマリー v0.2

入力: 00_案件設定.md（初回商談メモ・口頭テキスト）＋ ✋理解確認のヒアリング回答（模擬）

## 1. クライアント概要

- テクノブリッジ・ソリューションズ。中堅SIer、従業員約800名。官公庁・金融向けの受託開発が柱。
- 受託＝顧客の機密情報を預かる立場。セキュリティ・監査が事業の生命線。
- トップ（役員）から「生成AIで何とかしろ」という号令が出ている状況。

## 2. 相談の経緯

- 競合が「AIで開発生産性3割向上」と対外発表 → 役員から号令 → だが社として打ち手が定まらず相談に至った。
- 現場は個人レベルの利用（議事録清書程度）に留まり、組織的活用に至っていない。

## 3. 期待の構造化

| No | 期待（発言由来） | 性質 |
|---|---|---|
| W1 | PMの業務（議事録・進捗取りまとめ・提案書・レビュー準備）を効率化し、顧客折衝/意思決定に時間を回したい | 業務効率化 |
| W2 | 企画〜要件定義の立ち上げが遅い（数週間）。スピードを上げたい | リードタイム短縮 |
| W3 | PM間の生成AI活用レベルのばらつきを揃え、組織全体を底上げしたい | スキル平準化／組織能力 |
| W4 | 顧客機密・監査・セキュリティ統制を絶対に崩さない | 統制（前提条件・絶対） |
| W5 | 役員に説明できる形にしたい（"なんとなくAI入れた"では困る） | 説明責任／対経営 |

- **手段先行の混在**: 「生成AIで何とか」は手段。目的はW1〜W3（時間・スピード・組織能力）。AIは目的ではないと整理する。

## 4. 制約・条件（出どころ付き）

- 期間: 未定（出どころ②未定 → こちらが設計・提示する）
- 予算: 未定（出どころ②未定）
- 絶対条件: 顧客機密保護・監査/セキュリティ統制の維持（出どころ①クライアント指定。トレードオフ不可）

## 5. 事実と推測の区別

- 事実（発言で裏付け）: 役員号令がある／PM業務の逼迫／立ち上げが遅い／活用レベルにばらつき／統制は絶対条件／予算期間未定。
- 推測（要検証）: 「3割」競合発表の中身（何の3割か不明）／PMの時間がどの作業に何割奪われているかの内訳／統制要件の具体（どの規程・監査基準か）／意思決定の実権の所在。

## 6. 不明点 → ヒアリング質問（優先度順）

1. 【最大リスク】崩してはいけない統制の具体は？（準拠規程・監査基準・既存の承認/ログ運用、AIツール利用の社内規定の有無）
2. 過去にツール導入や全社施策で「現場がついてこなかった」失敗はあるか？
3. PMの時間は実際どの作業に何割か（議事録/ドキュメント/進捗/レビュー/折衝）。最も削りたいのはどれか
4. 成功イメージ。半年後どうなっていれば「やってよかった」か。役員が見たい指標は？
5. 意思決定と予算の実権は誰か。今回の窓口Aさんはどこまで決められるか

## 7. ヒアリング結果（v0.2で追加・模擬回答）

- Q1【談】準拠は主に金融系顧客のFISC安全対策基準と官公庁案件の調達セキュリティ要件。社内にAIツール利用規程は未整備（"野良利用"が実態で、むしろそこが不安）。
- Q2【談】3年前にプロジェクト管理ツールを全社導入したが、現場が入力せず形骸化した「失敗体験」がある。"また同じになるのでは"という警戒が強い。
- Q3【未】感覚では議事録・ドキュメント整備が最も重いが、定量データは持っていない（測っていない）。
- Q4【談】役員が見たいのは「立ち上げリードタイムの短縮」と「PM一人あたりの担当案件数」。まずは小さくても成功事例が欲しい。
- Q5【談】予算の最終決裁は担当役員。Aさんは数百万規模の試行までは起案できる。本格展開は役員説明が必須。

### ヒアリングから得た示唆

- **最重要の軸**: 「統制を崩さずに（W4）」が全打ち手の前提条件。効率化（W1）より優先する制約。エクスプラザの統制観点（権限・ログ・HITL）が刺さる。
- 過去のツール形骸化（Q2）は最大の阻害要因。「現場が入力しない/使わない」の再来が最大リスク。定着設計が肝。
- W5（役員説明）の評価軸は確定: 立ち上げリードタイム短縮 ＋ PM一人あたり担当案件数。成功基準はこの本人の言葉から作る。

### 「回答者が知らないこと」自体の発見

- **PM業務の時間内訳が測られていない（Q3【未】）** → 現状の定量化（ベースライン測定）自体を初期フェーズに組み込む根拠になる。「どこを削れば効くか」を当てずっぽうで決めない。
- **AIツール利用規程が未整備（Q1）** → 効率化の前に「統制の土台づくり（利用ガイドライン・権限・ログ）」が先行課題として浮上。
