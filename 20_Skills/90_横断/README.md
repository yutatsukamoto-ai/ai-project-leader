# 90_横断（横断Skillの枠）

`20_Skills` 配下のフェーズ別Skill（00_前段〜05_終結）に対して、**フェーズをまたいで効く"Skill"**を置く枠。

- ここに置くのは **実行可能なSkill束**（例: 将来の文章レビューSkill、作図補助Skillなど）。
- **横断ガイドラインの"ドキュメント"はここに置かない**。それらの正典は `40_Stock/横断ガイドライン/` にある（文章の書き方＝`文章作法.md` など）。

意味の混同（Skill束とドキュメント置き場）を避けるための区分。

## 状態：予約枠（現状 空）

ここが空なのは**異常ではなく予約**。フェーズをまたぐ実行Skill（例: 文章レビュー／作図補助）が生まれたら設置する。それまでは空のまま置く。

「横断」系の住み分け早見は `20_Skills/成果物マップ.md` の「横断・メタ」表が正典：

- 横断"実行Skill"の枠 → ここ `20_Skills/90_横断/`（予約・現状空）
- メタ（Skillを作るSkill） → `20_Skills/99_メタ/`
- 表現ルールの"ドキュメント"正典 → `40_Stock/横断ガイドライン/`
