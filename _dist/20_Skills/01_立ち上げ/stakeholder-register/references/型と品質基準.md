# ステークホルダー登録簿 型と品質基準

出典: 『プロジェクトリーダーの仕事』第4章（立ち上げ）／PMBOK第8版 #23 ステークホルダーの特定／12ステップ定義#3 利害関係者。
位置づけ: チャーター§7のハイレベル一覧を「誰を・どの関与度へ・どう動かすか」まで深掘りした生きた台帳。成否を分けるのは技術でなく巻き込み、を実装する成果物。

## 品質バー（このSkillの合格条件）

1. **意思決定者・キーパーソン・承認者**を漏れなく挙げている（#23／#3）。
2. 各関係者に **関与度（現在→目標）** と **巻き込み戦略** がある（一覧で終わらせない）。
3. **重点管理が絞れている**（影響力×関心で。広げすぎない）。
4. チャーター§7と**二重化でなく深掘り・接続**になっている。

## 登録簿の項目

| 項目 | 何を書くか |
|---|---|
| 関係者（名前・役割） | 個人・部門・外部。役割を併記 |
| 区分 | 内部／外部 |
| 影響力 | 高／中／低（意思決定・現場への効き） |
| 関心 | 高／中／低（この案件への当事者度） |
| 重点度 | 影響力×関心で4分類：**重点管理**（高×高）／**満足を保つ**（高×低）／**情報提供**（低×高）／**様子見**（低×低） |
| 関与度 現在→目標 | 〔不認識／抵抗／中立／支持／主導〕。**現在と目標のギャップが戦略を生む** |
| 巻き込み戦略 | ギャップを埋める一手。一言で |
| 連絡（頻度・手段） | 月1報告会・週次・随時 等（社内省略可） |
| 担当 | 誰が関係を持つか |
| 状態 | ✅確定／🟡未確定（指名未・人選未は確定タイミングを書く） |

## 判断の分かれ目（人が確認すべき点。Skillはここで止まる/問う）

- **二重化回避**: チャーター§7は"立ち戻る一枚"として残す。登録簿は深掘りの別物。重複は許すが、両方を正典化しない（リンクで繋ぐ）。
- **関与度は現在→目標で書く**: 「支持してほしい」でなく「いま中立→目標は支持」。ギャップが無ければ戦略は要らない。
- **重点管理を絞る**: 当方の時間と接触頻度は有限。最も成否を分ける数名に集中させる。全員に薄く当たらない。
- **抵抗者がキーパーソンを兼ねるとき**: 当事者化（痛みを入口に巻き込む）を第一候補に。ただし腹落ち前の名ばかり任命はしない（動かないから＝効果の理屈）。任命は腹落ちを確認するゲートで確定。
- **可逆／不可逆の線引き（最重要の停止点）**:
  - 可逆な賭け（巻き込み方・関与度の読み）→ 効果優先で攻める。読み違いは後から直す。
  - 不可逆・致命（情報漏洩・契約・安全・信頼の致命的破壊＝プロジェクトや今後のDXを殺す）→ 賭けず、必ず関所（✋✋）。遅くなっても通す。
- **🟡の扱い**: 指名未・人選未は空欄にせず🟡＋確定タイミング（どのゲートで決めるか）を書く。
