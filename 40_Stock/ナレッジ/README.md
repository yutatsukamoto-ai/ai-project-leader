# ナレッジ

このフォルダは、案件や勉強会から得た知見を、次回以降も使える形に再構成して置く場所。

## 置き方

- raw文字起こし、議事録、案件固有メモは `30_Flow/` に置く
- `40_Stock/ナレッジ/` には、個人名・社名・会話固有の文脈を除いた再利用版だけを置く
- 書籍由来の要点は引用集ではなく、自分の言葉で整理したメモとして置く
- 案件で得た反省や再発防止は、汎用化できるものだけ `40_Stock/案件教訓/` に分ける

## 主要ファイル

| ファイル | 用途 |
|---|---|
| `12ステップ要点.md` | プロジェクト推進の手順理解 |
| `PMBOK第8版_40プロセスと7領域.md` | PMBOK語彙の俯瞰 |
| `PoC遭難本_アンチパターン7選.md` | PoC案件で避けるべき落とし穴 |
| `image2-brand-slides_設計思想と使いどころ.md` | Image 2によるブランドスライド生成Skillの設計思想 |
| `AIスライド生成_アンチパターン.md` | AIスライド生成で避けるべき進め方 |
| `書籍要点/` | 書籍由来の再構成メモと参照ルール |
