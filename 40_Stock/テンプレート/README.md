# テンプレート（成果物に付随する記入フォーム・雛形）

成果物に付いてくる「**記入して使う空欄フォーム・雛形＋その処理ルール**」の正典置き場。
人の記入ガイドと、AIへの反映規約を1枚に持つ「テンプレ兼AIナレッジ」。

## 入れる / 入れない（住み分け）

- **入れる**：記入→AIが反映する型（例：WBS更新依頼フォーム）、各成果物に付く運用フォーム・空欄雛形。
- **入れない**：
  - 表現ルール（書き方の決まり）→ `40_Stock/横断ガイドライン/`
  - 勘所・症状処方（知識の小片）→ `40_Stock/カード/`
  - 書籍の再構成要点 → `40_Stock/ナレッジ/`
  - **案件ごとの"記入済み"インスタンス** → その案件フォルダ（Flow/Stock）。ここは空欄の正典だけ。

## 命名・成果物ごとの分け方

- **成果物名でまとめる**。命名は `{成果物}_{用途}テンプレ.md`（例：`WBS_更新依頼テンプレ.md`）。プレフィックスで成果物ごとに自然に固まる。
- **段階式**：1つのうちは平置き。**同じ成果物のテンプレが2つ以上に増えたら `{成果物}/` サブフォルダに束ねる**（例：`WBS/更新依頼テンプレ.md`・`WBS/初期入力テンプレ.md`）。薄いうちは平置き、増えたら畳む。
- どの成果物のものか一目で分かる状態を常に保つ。

## Skill・案件との関係

- ここが**正典**。必要なSkillは `references/` へ同梱、または sync マニフェストで配布（文章作法と同じ方式）。
- 案件では、この空欄をコピーして記入したものを案件フォルダに置く（正典は汚さない）。
- 一覧は `20_Skills/成果物マップ.md` から参照する。
